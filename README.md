Execute o arquivo script.sql no seu banco de dados MySQL.

Configure o arquivo conectar.php com as credenciais do seu ambiente.

Coloque o projeto no diretório do servidor local (ex: htdocs no XAMPP).

Acesse pelo navegador em http://localhost/bootcamp-entrega.

Estrutura principal
index.php              # Login
criar-conta.php         # Cadastro de usuário
listar-tarefas.php      # Listagem de tarefas
tarefa.php              # Criação/edição de tarefa
atualizar-senha.php     # Alteração de senha
conectar.php            # Conexão com o banco de dados
script.sql              # Script de criação do banco
